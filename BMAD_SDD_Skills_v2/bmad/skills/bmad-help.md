# Skill: bmad-help

## Purpose
Hướng dẫn bạn chạy BMAD theo phase hiện tại.

## Inputs
- phase: discovery|planning|solutioning|execution|quality

## Outputs
- danh sách skill cần chạy + file cần đọc/ghi

## Steps
- discovery → create_product_brief + create_sdd_master_spec
- solutioning → secure_connection_design + create_api_contract + create_ui_spec
- execution → implement_story_fe + review_story_fe
- quality → security_privacy_review + create_test_plan + release_checklist
