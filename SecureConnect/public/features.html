📋 Tổng quan hệ thống

DatabaseManager là gì?

DatabaseManager là ứng dụng web quản lý cơ sở dữ liệu an toàn, hỗ trợ múi database backends (MySQL, PostgreSQL, SQL Server). Mọi kết nối và dữ liệu được bảo vệ bằng mã hoá end-to-end.

👨‍💼 Hướng dẫn Admin & Deployment

Variable Example Required

AUTH_GOOGLE_SECRET GOCSPX-*** ✓

AUTH_ALLOW_ANY true / false ✗ (default: false)
ALLOWED_EMAILS user1@example.com, user2@example.com ✗
ALLOWED_EMAIL_DOMAINS example.com, company.org ✗

Docker Containers

Container Image Role
dbmanager-app dbmanager:latest Next.js app, port 9230
dbmanager-redis redis:7-alpine Session + schema cache, AOF persistence

Deployment Process

  1. Push code: git push origin main
  2. SSH to server: ssh root@mcp-80
  3. Pull & build: 
    cd /opt/dbmanager
git pull origin main
docker compose build --no-cache app
docker compose up -d --force-recreate app
  4. Verify: curl https://DBManager.allin1site.com/api/health

Monitoring & Logs

SSH vào server và chạy:

Command Mục đích
docker ps -a | grep dbmanager Kiểm tra container status
docker logs dbmanager-app --tail 50 Xem logs app
docker exec dbmanager-app npm run type-check Run TypeScript check
curl -s http://127.0.0.1:9230/api/health Health check local
redis-cli -h redis info stats Redis stats (schema cache)

Scaling & Resource Limits

App Memory: 512MB limit, 128MB reservation
App CPU: 1.0 core (full)
Redis Memory: 192MB limit
Redis Policy: volatile-ttl (evict expired keys)

Troubleshooting

Issue Diagnosis Fix
White screen on login DevTools console → JavaScript errors? Check code, rebuild, redeploy
OAuth error=Configuration Google credentials missing or wrong Check .env.production, restart container
502 Bad Gateway Check nginx → app port mapping Verify port 9230, restart nginx
Slow queries Redis cache miss? Large dataset? Check SQL EXPLAIN, optimize query or DB indexes
Redis OOM Cache growing unbounded? Check Redis memory, restart, increase TTL
